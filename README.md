# KWIK CHESS

KWIK CHESS is a browser based chess board training game.

It started as a quick way to practise square recognition, but it is growing into a small collection of fast chess board challenges for memory, movement, piece placement and visual spotting.

The project is currently in beta. I am still adding modes, tuning the layout, improving mobile play and making the leaderboards more useful.

## Play

Play KWIK CHESS at:

**https://kwikchess.com**

## Game modes

- **30s Sprint** — click as many called squares as possible in 30 seconds.
- **64 Squares** — find every square on the board as quickly as you can.
- **King Finder / King Finder XTREME** — remember where the kings were placed, then find them after the board clears.
- **Pick Up / Pick Up XTREME** — place the chess pieces back onto their correct starting squares. XTREME adds legal-move restrictions.
- **Knight Hop / Knight Hop XTREME** — move a knight around the board. XTREME is a knight’s tour-style challenge where every square must be visited once.
- **Odd One Out / Odd One Out Rainbow** — spot the square that does not match. It becomes harder as you level up.

## Why I made it

I wanted something simple, fast and slightly addictive for practising board vision without needing a full chess game.

Most modes can be played in short bursts. Some are useful for learning square names, some are about memory, and some are just there because they are fun.

## Status

KWIK CHESS is in beta.

Things may change quickly. Scores, modes, layout and difficulty may be adjusted as the game develops.

## Built with

KWIK CHESS is built as a lightweight web app using HTML, CSS and JavaScript.
